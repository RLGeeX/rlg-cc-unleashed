# Product Management Agent Dispatcher

Load a specialized product management agent for planning, analysis, and project management.

**Available Agents:**
- `analyst` - Business analyst (agents/product-management/business-analyst.md)
- `owner` - Product owner (agents/product-management/product-owner.md)
- `story` - Story writer (agents/product-management/story-writer.md)
- `jira` - Jira specialist (agents/product-management/jira-specialist.md)
- `scrum` - Scrum master (agents/product-management/scrum-master.md)

**Usage:**
- `/rlg-pm story` - Load story writer
- `/rlg-pm analyst` - Load business analyst
- `/rlg-pm` - Show available PM agents

If no agent is specified, present the list of available agents and ask the user which one to load.
