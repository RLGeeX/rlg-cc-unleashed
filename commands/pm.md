# Product Management Agent Dispatcher

Load a specialized product management agent for planning, analysis, and project management.

**Available Agents:**
- `business-analyst` - Business analyst
- `product-owner` - Product owner
- `story-writer` - Story writer
- `jira-specialist` - Jira specialist
- `scrum-master` - Scrum master

**Usage:**
- `/cc-unleashed:pm story-writer` - Load story writer
- `/cc-unleashed:pm business-analyst` - Load business analyst
- `/cc-unleashed:pm` - Show available PM agents

If no agent is specified, present the list of available agents and ask the user which one to load.
