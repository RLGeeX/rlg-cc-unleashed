Execute spec-kit tasks autonomously with constitution validation, agent dispatch, Jira sync, and code review gates.

Use the sk-execute skill to run through spec-kit tasks with intelligent agent selection, Jira status transitions, and mandatory code review between phases.

**Prerequisites:** Run `/cc-unleashed:sk-jira` first to create Jira issues and sk-state.json.

**Usage:** `/cc-unleashed:sk-execute <feature-path>`

**Example:** `/cc-unleashed:sk-execute specs/001-user-auth`
