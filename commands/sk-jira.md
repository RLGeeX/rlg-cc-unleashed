Create Jira Epic → Stories → Sub-tasks hierarchy from spec-kit artifacts (spec.md, tasks.md).

Use the sk-jira skill to create a complete Jira issue hierarchy based on spec-kit user stories and tasks. Creates sk-state.json for execution tracking.

**Usage:** `/cc-unleashed:sk-jira <feature-path>`

**Example:** `/cc-unleashed:sk-jira specs/001-user-auth`
