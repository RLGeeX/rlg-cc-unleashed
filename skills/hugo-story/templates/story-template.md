## Writing Guidelines (Remove Before Publishing)

### Frontmatter Requirements
- `title`: Engaging, includes main keyword, 50-70 chars
- `date`: Publication date in YYYY-MM-DD format
- `description`: Compelling meta description, 150-160 chars
- `featured_image`: Path to image or placeholder.svg
- `excerpt`: Short hook for article listings, 80-100 chars
- `author`: Always "Chris Fogarty"
- `categories`: 1-2 broad categories
- `tags`: 3-6 specific, searchable keywords

### Content Requirements
- **Length**: 800-2000 words depending on topic depth
- **Structure**: Introduction + 3-5 main sections + Conclusion
- **Headers**: Use H2 (##) for main sections, H3 (###) for subsections
- **Paragraphs**: 2-4 sentences each
- **Reading Level**: Grade 10-12 (accessible but professional)

### Style Guidelines
- **Voice**: Active voice preferred
- **Tone**: Professional but approachable
- **Technical Terms**: Explain or link to definitions
- **Examples**: Include concrete, real-world examples
- **Data**: Support claims with evidence where possible

### SEO Checklist
- [ ] Title includes main keyword
- [ ] Description is compelling and includes keywords
- [ ] Headers contain relevant terms
- [ ] Content is comprehensive for the topic
- [ ] Internal links where appropriate
